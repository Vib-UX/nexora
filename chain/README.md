# chain/

Orbit chain config + scripts for **Nexora Devnet** (chainId `20056` / `0x4E58`).

## Files

| File | Purpose |
|---|---|
| `orbit-config.json` | Canonical chain definition (consumed by deploy script + dashboard) |
| `deploy-l3.ts` | `plan` / `local` / `execute` modes for Orbit deployment |
| `docker-compose.yml` | Local Nitro node + optional Blockscout explorer |
| `data/nitro-node-config.json` | Generated by `mode=local`, mounted into Nitro container |

## Modes

```bash
# 1. Print plan only (no side effects)
pnpm --filter @nexora/chain deploy:l3

# 2. Generate a local nitro-node config for docker-compose
ORBIT_DEPLOY_MODE=local pnpm --filter @nexora/chain deploy:l3

# 3. Deploy the rollup contracts on parent chain (requires .env)
ORBIT_DEPLOY_MODE=execute pnpm --filter @nexora/chain deploy:l3
```

## Required environment (`.env` here)

```
PARENT_RPC_URL=https://sepolia-rollup.arbitrum.io/rpc
ORBIT_OWNER_PRIVATE_KEY=0xabc...
SEQUENCER_ADDRESS=0x...
VALIDATOR_ADDRESS=0x...
```

## Boot the local devnet

```bash
ORBIT_DEPLOY_MODE=local pnpm --filter @nexora/chain deploy:l3
docker compose up -d nitro
# optional explorer
docker compose --profile explorer up -d
```

RPC: `http://localhost:8547` · WS: `ws://localhost:8548` · Explorer: `http://localhost:4000`
